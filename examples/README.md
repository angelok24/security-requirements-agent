# Примеры

Реальные архитектурные схемы и наборы требований анализируемых систем являются **корпоративными данными** и в репозиторий не включаются.

Здесь приведён только **обезличенный иллюстративный пример**:

- `architecture_example.svg` — обобщённая архитектурная схема (внешние системы, IAM, сервис-сборщик, БД, дашборд), показывающая, какого рода изображение подаётся агенту на вход.

Результаты работы агента (метрики Precision / Recall / F1, сравнение моделей) приведены в основном [README](../README.md).
